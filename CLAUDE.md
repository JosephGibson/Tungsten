# CLAUDE.md

This project uses **`AGENTS.md`** as the canonical instruction file for AI coding assistants. Claude Code reads `CLAUDE.md` by convention; this file exists only to redirect to the actual instructions.

**Read `AGENTS.md`** for operational rules, repo layout, commands, conventions, and what not to do.

**Read `README.md`** for the recommended document reading order.

**Read `DESIGN.md`** for architecture and milestone context.

**Read `DECISIONS.md`** for the rationale behind specific non-obvious choices (append-only log).

The instructions in `AGENTS.md` apply whether you're Claude Code, Cursor, Codex, Aider, or any other tool. Keeping a single canonical file avoids drift between tool-specific copies.
